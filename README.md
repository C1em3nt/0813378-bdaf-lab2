# BDAF02 Vanity Address Generator

clone from github
``` 
git clone [https://github.com/d3or/eth-vanity-gen.git](https://github.com/C1em3nt/BDAF02.git)
```
Install dependencies

```
npm install
```

Run

```
npm start
```
Type the prefix you want.
